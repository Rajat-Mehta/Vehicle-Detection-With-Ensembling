This file contains list of required libraries which needs to be installed before running any of the script in the project.

1. Name: Python, Version: 3.6
2. Name: matplotlib, Version: 2.2.3
3. Name: numpy, Version: 1.13.1
4. Name: opencv-python, Version: 3.4.1
5. Name: Gluoncv, Version: 0.3.0, Command: pip install gluoncv==0.3.0
6. Name: mxnet, Version: 1.3.0 with CUDA version 8, Command: pip install mxnet-cu80
7. Name: pandas, Command: pip install pandas
