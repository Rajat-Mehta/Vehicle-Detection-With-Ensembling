This folder contains the result files and some sample result images.

1. class_wise_results: this file contains results of ensemble models including model averaging, weighted average and adaptive weights based ensemble model. It also includes the class wise average precisions for all of these models.

2. Final_Results.ods: this file contains the same information as in class_wise_results file but in compact tabular format.

3. sample_result_1 and _2: these folders contain detection outputs plotted on images.
